ℹ️ System Overview

🎯 Purpose

This system answers clinical coding questions and analyzes patient encounters for E/M code support, documentation gaps, and JAWDA compliance.

Supported Authorities

AMA 2021
E/M code selection (99202-99215), MDM 2-of-3 rule, time-based coding
CMS 1997
Documentation framework (HPI, ROS, PFSH, Exam), history level validation
HAAD
UAE-specific coding and documentation guidance
JAWDA 2026
Audit compliance, findings, scoring, and verification logic

Answer Quality Rules

  • ✅ Exact Citations: Every answer includes source, section, and page
  • ✅ Authority Routing: Questions routed to appropriate guideline
  • ✅ AMA 2021 Supersession: AMA overrides CMS for codes 99202-99215
  • ✅ No Hallucination: Answers only from stored guidelines
  • ✅ Structured Responses: Deterministic outputs with clear fields

How It Works

For Questions:

  1. System detects keywords and CPT codes in your question
  2. Routes to the most relevant authority (AMA/CMS/HAAD/JAWDA)
  3. Retrieves matching guideline sections
  4. Ranks results by relevance and returns top match with citation

For Encounters:

  1. System infers MDM level from documentation (problems, data, risk)
  2. Calculates recommended CPT code using 2-of-3 rule
  3. Compares with billed code
  4. Detects documentation gaps and JAWDA audit issues
  5. Calculates denial risk (low/moderate/high)
